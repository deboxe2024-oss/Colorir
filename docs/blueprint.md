# **App Name**: Conect Kids Devotionals

## Core Features:

- Devotional Content Delivery: Provide access to daily devotionals through a secure portal with login and password.
- Content Updates: Deliver new devotionals monthly until a full year (365 days) of content is available.
- Printable Material: Offer downloadable and printable files in high quality.
- Interactive Questions: Present thought-provoking questions to encourage children to understand and apply the daily theme. An LLM is used as a tool, and is given the verses and reflective sections from each day's devotionals, to ensure questions align with the lesson. This ensures consistency, while reducing developer workload
- User Support: Provide a direct line to the support team via WhatsApp for assistance and inquiries.

## Style Guidelines:

- Primary color: A warm yellow (#FFC107) to evoke feelings of happiness and energy.
- Background color: Light beige (#F5F5DC), providing a soft and inviting backdrop.
- Accent color: A gentle orange (#FF7043), complementing the primary yellow and enhancing important interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif with a balance of modern style and warmth. It suits both headlines and body text.
- Use clear, friendly, and age-appropriate icons that reflect themes of faith, family, and learning. These should be simple and easily recognizable for young children.
- Design a clean and straightforward layout that's easy for parents to navigate. Use clear sections for daily devotionals, interactive questions, and memory verses.
- Subtle animations on interactive elements to engage children. Use simple transitions between devotional components.
